World Wide Weather

A Source For Weather Across the World

As a full stack project, World Wide Weather uses a Python Flask server with a MySQL database to serve up real weather data from across the world. Local APIs are used to fetch the data asynchronously after page load, optimizing load times for the user. The site features the ability to change units of measure and create a dashboard of the current weather of a collection of cities through the creation of an account.


Check out World Wide Weather Now!

Messenger

A Realtime Group Messaging Service

Messenger implements React.js and Socket.io to create a dynamic two page application. Running an Express server, the project utilizes a series of API routes which interface with a MongoDB database and trigger Socket.io events to keep all clients updated in realtime. In addition, the use of the Material-UI library gives users a familiar and easy-to-use experience in addition to it's speed.


Check out Messenger now!

Capstone Virtual Event

A Highlight of Capstone Projects

The Capstone Virtual Event implements front-end technologies to provide an engaging and visually appealing platform to showcase graduating students capstone projects. The site features a brightly-colored vertical carousel home page and a show mode for watching the event using the YouTube API.
